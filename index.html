HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Is what the browser renders files down to. (In Chrome I think it's google's V8 Chromium Engine :3)

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • To link a CSS file to an HTML file, use the link element in the head element.
  • The CSS box model is a box that wraps around every HTML element. It consists of: content, padding, border, and margin.
